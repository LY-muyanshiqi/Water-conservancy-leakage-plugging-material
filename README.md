# 治水先行：土石堤坝渗漏封堵材料研发

## 项目概述
- **研究目标**：研发适用于水利工程应急抢险的封堵材料，解决"快凝与高强度无法兼顾"的核心矛盾。
- **技术路线**：以水泥基材料为基础，掺入粉煤灰/矿渣等固废，通过聚羧酸减水剂、纳米材料、海藻酸钠等添加剂实现性能调控。
- **实验规划**：9组正交配比变量，测试流动度、凝结时间、抗压强度、渗透系数等指标，构建配比-性能数据库。
